# fastapi-app

fastapi-app は FastAPI でプロジェクトを立ち上げる際のテンプレートです

# Installation

```bash
docker-compose up -d
poetry install
```

# Usage

```bash
git clone https://github.com/koshihoge/fastapi-app
cd fastapi-app
```

Press F5
